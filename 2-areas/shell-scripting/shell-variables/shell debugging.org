** Debugging with =-x=
:PROPERTIES:
:CUSTOM_ID: debugging-with--x
:END:
Print each line of the script by adding =-v= after the declaration of
the shell

#+begin_src sh
#!/bin/bash -v
#+end_src

Print the result of each line of the script by adding =-x= after the
declaration of the shell=.=

#+begin_src sh
#!/bin/bash -x
#+end_src
