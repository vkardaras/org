## What is Karate

> Karate is an open-source unified test automation platform combining API testing, API performance testing, API mocks & UI testing

### Features
- Simplified Syntax
- Built-in Test Doubles
- Data-Driven Testing
- Powerful Assertions
- Parallel Execution

### Tech Stack
- Built using Java
- Cucumber-JVM
- Apache HttpClient
- JSON and XML Support
- Gradle and Maven
- JavaScript Engine
