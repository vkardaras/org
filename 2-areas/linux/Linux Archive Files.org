Archive files

#+begin_src sh
# create an archive
tar -cf test.tar file1 file2 file3

# view info of an archive
ls -ltr test.tar

# view the contents of a tarball
tar -tf test.tar

# extract the contents of a tarball
tar -xf test.tar

# create and compress an archive
tar -zcf test.tar file1 file2 file3
#+end_src
