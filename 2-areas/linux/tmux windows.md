Windows

```bash
# rename window
<leader>,

# move to the next window
<leader>n

# move to the previous window
<leader>p

# swap windows (3 and 1)swap
<leader>: swap-window -s 3 -t 1

# swap current window with another (1)
<leader>: swap-window -t 1

# kill window
<leader>&
```
