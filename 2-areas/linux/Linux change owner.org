*** Kinds of permissions
:PROPERTIES:
:CUSTOM_ID: kinds-of-permissions
:END:
| Letters | Definition                                                                  |
|---------+-----------------------------------------------------------------------------|
| 'r'     | "read" the file's contents.                                                 |
| 'w'     | "write", or modify, the file's contents.                                    |
| 'x'     | "execute" the file. This permission is given only if the file is a program. |
