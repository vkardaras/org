* maven project archetype

** General archetype

#+begin_src sh
mvn archetype:generate
#+end_src

** Make a maven eclipse project from a general maven project

1. Move to the project generated with the general archetype
2. Enter the following command

#+begin_src sh
mvn eclipse:eclipse
#+end_src
