\section{Display interpreted code}
To write a fraction, you type $\frac{1}{2}$

\section{Display code without to interpret it}
To write a fraction, you type \verb+\frac{1}{2}+

To write a fraction, you type \verb|\frac{1}{2}|

To write a fraction, you type \verb?\frac{1}{2}?

\section{Write a code block}
\begin{verbatim}
for num in range(1, 40):
	print(num ** 2)
\end{verbatim}
